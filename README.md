# Event-Planning
- Develop Your Event Goal and Objectives.
- Organize Your Team.
- Establish Your Budget.
- Set the Date.
- Create an Event Master Plan.
- Book Your Venue.
- Brand Your Event.
- Identify and Establish Partnerships &amp; Sponsors.
- Create a Publicity Plan.
- Determine Day-Of Processes.
